# Музыка для игры

В эту папку нужно поместить аудиофайл:

**Имя файла:** `spln-novye-lyudi.mp3`

## Как добавить музыку:

1. Скачайте песню "Сплин - Новые люди"
2. Переименуйте файл в `spln-novye-lyudi.mp3`
3. Поместите его в эту папку: `public/music/`

## Структура папки:

```
public/
└── music/
    ├── README.md
    └── spln-novye-lyudi.mp3  (нужно добавить)
```

После добавления файла музыка автоматически начнет играть при открытии игры.

**Примечание:** Убедитесь, что файл имеет формат MP3 и название точно совпадает с указанным выше.
